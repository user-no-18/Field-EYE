# Field-EYE (Trace Lite)

A premium, modern white-themed Glassmorphism Admin Dashboard for field worker location tracking, travel history visualization, and budget spending analytics. Built with Next.js, React Leaflet, and Recharts.

## 🚀 Quick Start

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** Vanilla CSS (Glassmorphism & Responsive Layouts)
- **Maps:** Leaflet & React Leaflet (Custom markers, live tracking)
- **Charts:** Recharts (Worker expense & analytics visualizations)
- **Icons:** Lucide React

## 📂 Project Structure

```text
├── public/           # Static assets (custom avatars, icons)
├── src/
│   ├── app/          # App router pages, layout, and global styling
│   ├── components/   # Reusable UI components (LiveMap, ExpenseCharts, WorkerCard)
│   └── lib/          # Mock data and utility functions
└── package.json      # Dependencies and scripts
```
